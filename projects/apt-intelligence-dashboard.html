APT Intelligence Dashboard

Real-time threat intelligence dashboard that aggregates and visualizes Advanced Persistent Threat (APT) data from multiple public sources-maps, timelines, actor dossiers, and linked reports.

Screenshots

From the upstream repository (same visuals as GitHub README).

Installation

No build step. Clone or download the repository, then serve files over HTTP (recommended for live data and caching).

git clone https://github.com/michaelelizarov/apt-intelligence-dashboard.git
cd apt-intelligence-dashboard
python -m http.server 8000

Run

Open the app in your browser. You can also open index.html directly for offline-style use with cached data.

# Then visit:
http://localhost:8000
