ziochub (ZIoCHub)

IOC and YARA management platform for SOC teams: SQLite-backed storage, plain-text feeds for security devices, optional MISP and Cisco ESA integrations, and an offline-friendly deployment story.

Interactive demo

Open the browser-hosted ZIoCHub vDEMO to click through tabs, sample data, and UI flows-no install required. For production deployment and integrations, use the GitHub repository below.

Installation

On Linux with internet (from upstream README). Air-gapped flow uses package_offline.sh-see OFFLINE.md on GitHub.

git clone https://github.com/Zwerd/ziochub.git
cd ziochub
sudo ./setup.sh

Run

After setup, the installer configures services; day-to-day access is via the web UI on your configured host and port (see repo docs for systemd and HTTPS).

# Offline bundle example (on target server):
# unzip ziochub_installer.zip -d ziochub_install
# cd ziochub_install && sudo ./setup.sh --offline
