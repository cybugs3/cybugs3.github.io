                    Available Feed Endpoints
                
    
                
            
    
            
    Use these URLs to integrate ZIoCHub feeds into your security devices. All feeds serve only active (non-expired) IOCs in plain text.
    
            
    

                
                
    
                    
    
                        📄 Standard (Plain Text)
                    
    
                    
    Newline-separated list of IOC values. Compatible with most firewalls and SIEMs.
    
                    
    
                        
    /feed/ipIP Addresses
    
                        
    /feed/domainDomains
    
                        
    /feed/urlURLs
    
                        
    /feed/hashAll Hashes
    
                        
    /feed/md5MD5 only
    
                        
    /feed/sha1SHA-1 only
    
                        
    /feed/sha256SHA-256 only
    
                        
    /feed/emailEmail Addresses
    
                    
    
                
    

                
                
    
                    
    
                        🔥 Palo Alto EDL
                    
    
                    
    External Dynamic List format. URLs are served without the http(s):// protocol prefix.
    
                    
    
                        
    /feed/pa/ipIP Addresses
    
                        
    /feed/pa/domainDomains
    
                        
    /feed/pa/urlURLs (no protocol)
    
                        
    /feed/pa/md5MD5
    
                        
    /feed/pa/sha256SHA-256
    
                        
    /feed/pa/emailEmail Addresses
    
                    
    
                
    

                
                
    
                    
    
                        🛡️ Check Point (CSV)
                    
    
                    
    CSV format compatible with Check Point IoC feed import.
    
                    
    
                        
    /feed/cp/ipIP Addresses
    
                        
    /feed/cp/domainDomains
    
                        
    /feed/cp/urlURLs
    
                        
    /feed/cp/hashMD5, SHA-1, SHA-256, SHA-512 (all hashes)
    
                        
    /feed/cp/md5MD5 only
    
                        
    /feed/cp/sha1SHA-1 only
    
                        
    /feed/cp/sha256SHA-256 only (SHA-2)
    
                    
    
                
    

                
                
    
                    
    
                        📧 Cisco ESA
                    
    
                    
    Comma-separated email list for Cisco Email Security Appliance content filters.
    
                    
    
                        
    /feed/esa/emailEmail Addresses (CSV)
    
                    
    
                
    

                
                
    
                    
    
                        ☣️ Trellix ePO
                    
    
                    
    CSV hash feeds grouped by Ticket ID for Trellix Endpoint Security / ePO policy import.
    
                    
    
                        
    /feed/epo/files-listList of Ticket IDs
    
                        
    /feed/epo/<ticket_id>Hashes for specific ticket
    
                    
    
                
    

                
                
    
                    
    
                        📦 TAXII / STIX 2.x
                    
    
                    
    
                    
                    
    
                        
    /taxii2/Discovery
    
                        
    /feed/stixAll types (JSON bundle)
    
                    
    
                
    

                
                
    
                    
    
                        📜 YARA Rules
                    
    
                    
    List and download approved YARA rule files for EDR / sandbox integration.
    
                    
    
                        
    /feed/yara-listAll .yar filenames
    
                        
    /feed/yara-content/<file>Raw rule content

                    TAXII 2.1 Server Endpoints
                
    
                
            
    
            
    ZIoCHub exposes active IOCs as STIX 2.1 indicators over TAXII 2.1. Use these URLs in TAXII clients (e.g. Cisco IronPort ESA, ThreatConnect). Send Accept: application/taxii+json;version=2.1.
    
            
    
                
    
                    
    Server information
    
                    
    
                        
    /taxii2/Discovery
    
                        
    /taxii2/ziochub/API Root
    
                    
    
                
    
                
    
                    
    Collections
    
                    
    
                        
    /taxii2/ziochub/collections/List collections
    
                        
    /taxii2/ziochub/collections/indicators/Collection: indicators
    
                    
    
                
    
                
    
                    
    Objects & manifest
    
                    
    
                        
    /taxii2/ziochub/collections/indicators/objects/Get Objects (paginated)
    
                        
    /taxii2/ziochub/collections/indicators/manifest/Get Manifests
    
                        
    /taxii2/ziochub/collections/indicators/objects/<id>/Get an Object
    
                        
    /taxii2/ziochub/collections/indicators/objects/<id>/versions/Get Object Versions
