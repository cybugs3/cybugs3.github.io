LinkedIn OSINT Toolkit

Python toolkit for company discovery, employee scraping, role classification, and org-chart visualization-wired through a single funnel script and a standalone HTML viewer.

Screenshots

Prerequisites

Python 3.8+, Firefox, and network access for Selenium (geckodriver is resolved via webdriver-manager at runtime).

Installation

git clone https://github.com/michaelelizarov/linkedin-osint-toolkit.git
cd linkedin-osint-toolkit
python -m venv venv

# Linux / macOS
source venv/bin/activate
# Windows (PowerShell)
# .\venv\Scripts\Activate.ps1

pip install -r requirements.txt
cp .env.example .env
# Edit .env - set LINKEDIN_EMAIL and LINKEDIN_PASSWORD (see repo docs).

Run

Example: full funnel with one browser session (approve OTP on your device when prompted).

python src/osint_funnel.py --geo-code 103644278 --keyword "cybersecurity" --limit 5

Visualize JSON output in the browser:

# Open in a browser (file:// or via a static server):
src/org_chart_viewer.html
